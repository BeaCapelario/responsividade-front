Este repositório é dedicado ao estudo e aplicação de técnicas de design responsivo. O projeto está sendo desenvolvido com o objetivo de criar interfaces que se adaptam a diferentes tamanhos de tela, desde dispositivos móveis até desktops, utilizando as melhores práticas de CSS e organização de layout.

---

### O que está sendo explorado:
* Media Queries: Implementação de pontos de interrupção (breakpoints) para ajuste de layout.
* Flexbox e CSS Grid: Uso de modelos de layout flexíveis e bidimensionais para posicionamento dinâmico.
* Unidades Relativas: Aplicação de unidades como %, em, rem, vw e vh em substituição aos valores fixos.
* Mobile-First: Estratégia de desenvolvimento focada inicialmente em dispositivos menores.
* Imagens Flexíveis: Técnicas para garantir que elementos visuais acompanhem o redimensionamento do container.
